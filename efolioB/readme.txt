O programa foi desenvolvido em linguagem Prolog com a versão swi-prolog 8.4.2 e
testado no sistema operativo Windows 10

**************
* Compilação *
**************

Para compilar o programa basta escrever na linha de comandos, no diretorio do ficheiro, o seguinte comando:


   swipl --goal=main --stand_alone=true -o corposCelestes -c corposCelestes.pl           //Compilador - Cria Ficheiro Executável com base no código-fonte

**************
*  Execução  *
**************

Para executar o programa na versão compilada basta seguir os seguintes passos na consola:

   .\corposCelestes.exe
